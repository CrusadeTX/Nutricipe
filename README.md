# Nutricipe
Nutrition Management Web App
<h4>Overview</h4>
<p>Nutricipe is a nutrition management web application. It gives its users the ability to retrieve various food recipes based on the ingredients they have at home. Every user has the
ability to pick a Diet plan when they register, which illustrates their current nutrition goals and health hazards and characteristics. When a user adds a product to their personal "fridge" and comences a recipe search, only the recipes complying with the available products and diet characteristics are shown.
The application recognises two different kinds of users - "Administrators" and "Regular Users". Regular Users have access to the main functionalities of the application, Administrators also have access to the core functionalities and have the ability to manage users</p>
<h4>Used Technologies:</h4>
<h5>Front End:</h5>
<p>HTML, CSS, Bootstrap, JavaScript, JQuery, AJAX </p>
<h5>Back End:</h5>
<p>Java, Spring Boot, Spring Security<p>
<h5>Persistence</h5>
<p>The project uses the ORM Hibernate and H2 as the database engine</p>
<h5>Quality Assurance</h5>
<p>A majority of the code has been unit tested with JUnit and external services have been mocked with Mockito. Selenium for java has been used for conducting ui tests</p>
